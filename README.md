## IoT 笔记
> 40 年前热衷于硬件的创客们成就了个人电脑辉煌年代，统一的闭源系统使得软件业高速发展。如今，热衷于开源硬件的创客们姗姗来临，五花八门的智能硬件层出不穷，又一个辉煌年代。我在这里做点记录。
